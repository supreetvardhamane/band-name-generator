# 🎸 Band Name Generator

A simple web app that generates a **random band name** using a combination of adjectives and nouns.
Click the button and get a new band name instantly.

Built while learning **Node.js, Express, and EJS**.

## 🚀 Features

* Random band name generation
* Built with **Express.js** and **EJS**
* Uses **EJS partials** for reusable header and footer
* Large arrays of adjectives and nouns for unique combinations

## 🛠 Tech Stack

HTML • CSS • JavaScript • Node.js • Express.js • EJS

## 📦 Installation

```bash
git clone https://github.com/supreetvardhamane/band-name-generator.git
cd band-name-generator
npm install
node index.js
```

Open in browser:

```
http://localhost:3000
```

## 💡 How It Works

When the user clicks the **Generate** button, the server selects a random adjective and noun from arrays and renders the result dynamically using **EJS**.

Example output:

```
Electric Tigers  
Cosmic Shadows  
Funky Wolves
```
